# cmux-lite Swift フロントエンド

`cmux-lite` は、cmux-tui プロトコル v6 以降のセッションに接続するダークテーマの AppKit/GhosttyKit フロントエンドです。選択中スクリーンの分割ツリー全体を描画し、表示中の各ターミナルペインに 1 つのバイトアタッチメントを維持します。

## キーボードショートカット

| ショートカット | 操作 |
| --- | --- |
| Command-D | アクティブペインを右に分割 |
| Command-Shift-D | アクティブペインを下に分割 |
| Command-T | アクティブペインにタブを作成 |
| Command-W | アクティブタブを閉じる（最後のタブならペインも閉じる） |
| Command-N | ワークスペースを作成 |
| Command-1…9 | アクティブペインのタブ 1…9 を選択 |
| Control-1…9 | スクリーン 1…9 を選択 |
| Command-Option-矢印 | ローカルのレイアウト形状を使って隣接ペインへフォーカス |
| Command-Control-矢印 | 対応する分割比率を 0.05 ずつ変更 |

ショートカットは 1 つの共有アクションテーブルを使用します。一致しないキーはフォーカス中のターミナルへ渡されます。
