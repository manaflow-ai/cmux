бһ“бҹ…бһ–бҹҒбһӣбһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„ macOS бһўбһ¶бһ…бһ“бһ№бһ„бһҹбһ»бҹҶбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҖбһ¶бһҡбһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бһёбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбҹ” бһ…бһ»бһ… **Open** бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҸбҹ”

## бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҸбҹ’бһҡбһјбһңбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ cmux?

бһҒбҹ’бһүбһ»бҹҶбһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ Claude Code бһ“бһ·бһ„ Codex бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбҹ” бһҒбҹ’бһүбһ»бҹҶбһ’бҹ’бһӣбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫ Ghostty бһҮбһ¶бһҳбһҪбһҷ split panes бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ бһҫбһҷбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбһҫбһҳбһҡбһ”бһҹбҹӢ macOS бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ№бһ„бһҗбһ¶бһ“бҹ…бһ–бҹҒбһӣбһҺбһ¶бһҠбҹӮбһӣ agent бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһүбһ»бҹҶбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҡбһ”бһҹбҹӢ Claude Code бһҸбҹӮбһ„бһҸбҹӮбһҹбһҡбһҹбҹҒбһҡбһҸбҹ’бһҡбһ№бһҳ "Claude бһңбһ¶бһҖбҹҶбһ–бһ»бһ„бһҡбһ„бҹӢбһ…бһ¶бҹҶбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһёбһўбҹ’бһ“бһҖ" бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ”бһҡбһ·бһ”бһ‘ (context) бһ бһҫбһҷбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бһҫбһҖ tab бһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһҖ бһҒбҹ’бһүбһ»бҹҶбһҹбһ№бһ„бһҸбҹӮбһҳбһ·бһ“бһўбһ¶бһ…бһўбһ¶бһ“бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһ”бһ¶бһ“бһ‘бҹҖбһҸбһ•бһ„бҹ”

бһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бҹ’бһҡбһҫ coding orchestrators бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһ–бһҪбһҖбһңбһ¶бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Electron/Tauri бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (performance) бһҡбһ”бһҹбҹӢбһңбһ¶бһҡбҹҶбһҒбһ¶бһ“бһҠбһӣбҹӢбһҒбҹ’бһүбһ»бҹҶбҹ” бһҳбҹ’бһҷбҹүбһ¶бһ„бһ‘бҹҖбһҸ бһҒбҹ’бһүбһ»бҹҶбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ”бҹ’бһҡбһҫ terminal бһҮбһ¶бһ„ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮ GUI orchestrators бһҸбҹӮбһ„бһҸбҹӮбһҖбҹҶбһҺбһҸбҹӢбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ (workflow) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ cmux бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбһҫбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ macOS бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Swift/AppKitбҹ” бһңбһ¶бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ libghostty бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү terminal бһ“бһ·бһ„бһўбһ¶бһ“ config бһҡбһ”бһҹбҹӢ Ghostty бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ themes, fonts бһ“бһ·бһ„бһ–бһҺбҹҢбҹ”

бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӮбһәбһҡбһ”бһ¶бһҡбһ…бҹҶбһ бҹҖбһ„ (sidebar) бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бҹ” бһҡбһ”бһ¶бһҡбһ…бҹҶбһ бҹҖбһ„бһҳбһ¶бһ“ tab бһ”бһүбҹ’бһҲбһҡбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһё git branch, бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–/бһӣбҹҒбһҒ PR, бһҗбһҸбһҖбһ¶бһҡбһ„бһ¶бһҡ, port бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҹбҹ’бһҸбһ¶бһ”бҹӢ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ workspace бһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ…бһ¶бһ”бҹӢбһҷбһҖ terminal sequences (OSC 9/99/777) бһ“бһ·бһ„бһҳбһ¶бһ“ CLI (`cmux notify`) бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢ agent hooks бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Claude Code, OpenCode бһҮбһ¶бһҠбһҫбһҳбҹ” бһ“бҹ…бһ–бҹҒбһӣ agent бһҖбҹҶбһ–бһ»бһ„бһҡбһ„бҹӢбһ…бһ¶бҹҶ бһ•бҹ’бһ‘бһ¶бҹҶбһ„ (pane) бһҡбһ”бһҹбҹӢбһңбһ¶бһ“бһ№бһ„бһҳбһ¶бһ“бһҡбһ„бҹ’бһңбһ„бҹӢбһ–бһҺбҹҢбһҒбҹҖбһң бһ бһҫбһҷ tab бһ“бһ№бһ„бһ—бҹ’бһӣбһәбһЎбһҫбһ„бһ“бҹ…бһӣбһҫбһҡбһ”бһ¶бһҡбһ…бҹҶбһ бҹҖбһ„ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһҠбһ№бһ„бһҗбһ¶бһҳбһҪбһҷбһҺбһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһүбһ»бҹҶбһ“бҹ…бһ‘бһјбһ‘бһ¶бҹҶбһ„ splits бһ“бһ·бһ„ tabs бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһ…бһ»бһ… Cmd+Shift+U бһҠбһҫбһҳбҹ’бһ”бһёбһӣбҹ„бһҸбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбһ¶бһҡбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһўбһ¶бһ“бһҗбҹ’бһҳбһёбһ”бҹҶбһ•бһ»бһҸбҹ”

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё (in-app browser) бһҳбһ¶бһ“ scriptable API бһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбһҖбһ…бҹҒбһүбһ–бһё [agent-browser](https://github.com/vercel-labs/agent-browser)бҹ” Agents бһўбһ¶бһ…бһҗбһҸбһ…бһҳбҹ’бһӣбһ„ (snapshot) бһҠбһҫбһҳбһҲбһҫбһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ (accessibility tree), бһҷбһҖ element refs, бһ…бһ»бһ… (click), бһ”бҹҶбһ–бҹҒбһүбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ”бҹӮбһ”бһ”бһ‘ (fill forms) бһ“бһ·бһ„бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ (evaluate) JSбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹҶбһ”бҹӮбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ“бҹ…бһҖбҹ’бһ”бҹӮбһҡ terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһұбҹ’бһҷ Claude Code бһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҮбһ¶бһҳбһҪбһҷ dev server бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһўбҹ’бһңбһёбҹ—бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡ script бһ”бһ¶бһ“бһҸбһ¶бһҳбһҡбһҷбҹҲ CLI бһ“бһ·бһ„ socket API вҖ” бһ”бһ„бҹ’бһҖбһҫбһҸ workspaces/tabs, бһ”бҹҶбһ”бҹӮбһҖ panes, бһ”бһүбҹ’бһҮбһјбһ“ keystrokes, бһ”бһҫбһҖ URLs бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ”

## бһ‘бһҹбҹ’бһҹбһ“бһңбһ·бһҮбҹ’бһҮбһ¶бһҡбһ”бһҹбҹӢ cmux (The Zen of cmux)

cmux бһҳбһ·бһ“бһ”бһ„бҹ’бһҒбҹҶбһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһ‘бҹҒбҹ” бһңбһ¶бһӮбһәбһҮбһ¶ terminal бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбҹӮбһӣбһҳбһ¶бһ“ CLI бһ бһҫбһҷбһўбҹ’бһңбһёбҹ—бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһӮбһәбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһўбҹ’бһ“бһҖбҹ”

cmux бһӮбһәбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮ (primitive) бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ–бҹҒбһүбһӣбҹҒбһүбһ‘бҹҒбҹ” бһңбһ¶бһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ“бһјбһң terminal, бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ, бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„, workspaces, splits, tabs бһ“бһ·бһ„ CLI бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһңбһёбҹ—бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹҒбҹҮбҹ” cmux бһҳбһ·бһ“бһ”бһ„бҹ’бһҒбҹҶбһўбҹ’бһ“бһҖбһұбҹ’бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһңбһ¶бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ‘бһ»бһҖбһҳбһ»бһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ coding agents бһ“бҹ„бҹҮбһ‘бҹҒбҹ” бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһӮбһәбһҮбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбҹҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҸбҹӮбһ„бһҸбҹӮбһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бҹ” бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһҡбһҖбһғбһҫбһүбһңбһ·бһ’бһёбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ agents бһ“бҹ…бһЎбһҫбһҷбһ‘бҹҒ бһ бһҫбһҷбһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бһӣбһ·бһҸбһ•бһӣбһ”бһ·бһ‘бһҮбһ·бһҸ (closed products) бһҖбҹҸбһ…бҹ’бһ”бһ¶бһҹбҹӢбһҮбһ¶бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҡбһҖбһғбһҫбһүбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбҹӮбһӣбһҷбһӣбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһё codebases бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒ бһ“бһ№бһ„бһҡбһҖбһғбһҫбһүбһңбһ¶бһҳбһ»бһ“бһӮбҹҒбҹ”

бһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҳбһҪбһҷбһӣбһ¶бһ“бһ“бһ¶бһҖбҹӢбһ“бһјбһңбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһҠбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһҹбҹҶбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“ бһ“бҹ„бҹҮбһ–бһҪбһҖбһӮбҹҒбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бҹҶбһ•бһ»бһҸ бһӣбһҝбһ“бһҮбһ¶бһ„бһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡбһ•бһӣбһ·бһҸбһ•бһӣбһҺбһ¶бһҳбһҪбһҷбһўбһ¶бһ…бһҡбһ…бһ“бһ¶бһ–бһёбһӣбһҫбһ…бһ»бҹҮбһҖбҹ’бһҡбҹ„бһҳ (top-down) бһ‘бҹ…бһ‘бҹҖбһҸбҹ”

## бһҜбһҖбһҹбһ¶бһҡ (Documentation)

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ cmux, [бһҹбһјбһҳбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„](https://cmux.com/docs/getting-started?utm_source=readme)бҹ”

## бһҖбһ¶бһҡбһҹбҹ’бһҠбһ¶бһҡ Session бһЎбһҫбһ„бһңбһ·бһү (бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“)

бһ“бҹ…бһ–бҹҒбһӣбһ”бһҫбһҖбһЎбһҫбһ„бһңбһ·бһү бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ cmux бһ“бһ№бһ„бһҹбҹ’бһҠбһ¶бһҡбһҸбҹӮбһ”бҹ’бһӣбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹҒбһҸбһ¶ (metadata) бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ–

* бһ”бҹ’бһӣбһ„бҹӢ Window/workspace/pane
* бһҗбһҸбһҖбһ¶бһҡбһ„бһ¶бһҡ (Working directories)
* Terminal scrollback (бһ”бҹ’бһҡбһ№бһ„бһ”бҹ’бһҡбҹӮбһ„бһұбҹ’бһҷбһўбһҹбҹӢбһӣбһ‘бҹ’бһ’бһ—бһ¶бһ–)
* бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҡбһ»бһҖбһҡбһҖ бһ“бһ·бһ„ URL бһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ

cmux **бһҳбһ·бһ“** бһҹбҹ’бһҠбһ¶бһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢ (live process state) бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё terminal бһЎбһҫбһҷбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ session бһҡбһ”бһҹбҹӢ Claude Code/tmux/vim бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһўбһ¶бһ…бһ”бһ“бҹ’бһҸбһЎбһҫбһ„бһңбһ·бһүбһ”бһ¶бһ“бһ‘бҹҒбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһЎбһҫбһ„бһңбһ·бһүбҹ”
